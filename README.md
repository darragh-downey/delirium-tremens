### Getting Started

1. Clone the repository:

```bash
git clone https://github.com/oit-cyber/book.git
cd book
```

2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm dev
```

4. Build for production:

```bash
pnpm build
```
